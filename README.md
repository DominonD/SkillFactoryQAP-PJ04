# SkillFactoryQAP-PJ04
Public repository for SkillFactory homework

Данные тестовые сценарии осуществляют проверку работы продукта https://start.rt.ru/ по бриф-заданию от Ростелеком

Для запуска необходимо:

1) Установить необходимые библиотеки через консоль: pip3 install -r requirements

2) Скачать драйвер из: https://github.com/mozilla/geckodriver/releases

3) Запустить тесты командой: pytest --driver Firefox --driver-path /path/to/geckodriver.exe

В качестве инструмента для тестирования был использован сервис
https://internxt.com/ru/temporary-email
Этот сайт предлагает услугу создания одноразовой электронной почты,
что помогает создать тестовые данные для регистрации и авторизации.
Так как с брифом задания не прилагались тестовые данные пользователя,
с помощью данного сервиса были использованы 2 электронные почты
zzjj0@airsworld.net и 3xnu0@airsworld.net с целью создания 2 пользователей
для проверки форм регистрации и авторизации.

Также использовалась функция браузера "инструменты разработчика" для отслеживания
элементов страницы и статустного кода страницы. После ручного тестирования эти данные
использовались для написания автотестов.
